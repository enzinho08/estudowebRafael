O que é o html?

É uma linguagem de marcação usada para marcar e estruturar o conteúdo de uma página web

Seus conceitos básicos:

  1. Linguagem de marcação, não é considerado uma linguagem de programação
  2. Baseado em tags que definem o elemento
  3. Todos possuem sua extensão .html para leitura

A estrutura básica de um html

Todo documento HTML deve ser seguido por uma estrutura básica Começando com o conceitos chaves:

CONCEITOS CHAVES:

  1. <!DOCTYPE html>

    • Necessário para definir o tipo do documento como html5 e sempre ler ele como html



  2. <HTML>

    • Tag na qual envovle todo o conteúdo a ser acrescido na pagina, maybe a tag na qual o computador lerá para entendimento da linguagem e converterá a mesm! Perdura por todo o codigo, incluindo head e body que são inserido dentro dela, pois ambos serão displayados pela maquina por estarem dentro da tag html. Então sendo assim todo código deverá ser realizado na mesma, possui atributo para setagem de linguagem sempre utiliza <html lang "pt-br", ou idioma de sua escolha!!>



  3. <Tags e Elementos HTML>

    • Tags são utilizadas para criar elementos em HTML. Elas são escritas em colchetes angulares <> e geralmente vêm em pares (tag de abertura e tag de fechamento) <> e

    • Atributos fornecem informações adicionais sobre os elementos HTML, são sempre especificiados nas taga de abertura
